# tubes-oop-java-kelompok8
Aplikasi Pengelolaan Jadwal Kuliah untuk mengatur jadwal harian dengan fitur tambah mata kuliah berbasis objek, edit jadwal, dan reminder dengan perulangan waktu. Menggunakan GUI GridBagLayout berbentuk kalender, tombol reminder, drag-and-drop (opsional), serta modular package dan exception konflik waktu.
